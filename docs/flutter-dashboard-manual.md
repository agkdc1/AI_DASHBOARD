## 1. はじめに / 소개

### ダッシュボードとは / 대시보드란

シンビジャパン統合ダッシュボードは、3つの業務システムを1つの画面で操作できるアプリケーションです。

신비재팬 통합 대시보드는 3개의 업무 시스템을 하나의 화면에서 조작할 수 있는 애플리케이션입니다.

| システム / 시스템 | 役割 / 역할 | URL |
|---|---|---|
| InvenTree | 在庫・部品・注文管理 / 재고·부품·주문 관리 | portal.your-domain.com |
| Vikunja | タスク・プロジェクト管理 / 업무·프로젝트 관리 | tasks.your-domain.com |
| Outline | 社内Wiki・マニュアル / 사내 위키·매뉴얼 | wiki.your-domain.com |

### アクセス方法 / 접속 방법

- **Web**: `https://app.your-domain.com` をブラウザで開く / 브라우저에서 열기
- **Android**: 同じURLをChromeで開き、「ホーム画面に追加」/ 같은 URL을 Chrome에서 열고 "홈 화면에 추가"
- **対応ブラウザ / 지원 브라우저**: Chrome, Edge, Safari, Firefox

### 必要なもの / 필요한 것

- 会社のGoogleアカウント（`@your-domain.com`）/ 회사 Google 계정
- インターネット接続 / 인터넷 연결

## 2. ログイン / 로그인

### ログイン手順 / 로그인 절차

1. `https://app.your-domain.com` にアクセス / 접속
2. ログイン画面が表示される / 로그인 화면이 표시됨
3. **「Googleでログイン」** ボタンをクリック / **"Google로 로그인"** 버튼 클릭
4. 会社のGoogleアカウントを選択 / 회사 Google 계정 선택
5. 自動的に在庫タブ（ホーム画面）に遷移 / 자동으로 재고 탭 (홈 화면)으로 이동

### 認証の仕組み / 인증 구조

- Googleアカウントで一度ログインすると、3つのシステム（InvenTree、Vikunja、Outline）すべてに同時にアクセスできます。
- Google 계정으로 한 번 로그인하면 3개 시스템 (InvenTree, Vikunja, Outline) 모두에 동시 접속됩니다.
- 個別のパスワードは不要です / 별도의 비밀번호는 필요 없습니다.

### ログアウト / 로그아웃

1. 画面下部の **「設定」** タブをタップ / 화면 하단 **"설정"** 탭 클릭
2. 一番下の **「ログアウト」** をタップ / 맨 아래 **"로그아웃"** 클릭
3. ログイン画面に戻ります / 로그인 화면으로 돌아갑니다

## 3. ダッシュボードの構成 / 대시보드 구성

画面下部に4つのタブがあります:

화면 하단에 4개의 탭이 있습니다:

| タブ / 탭 | アイコン | 機能 / 기능 |
|---|---|---|
| **在庫 / 재고** | 📦 | 部品、在庫、発注、受注、送り状 / 부품, 재고, 발주, 수주, 운송장 |
| **タスク / 작업** | ✅ | プロジェクト、カンバン、カレンダー / 프로젝트, 칸반, 캘린더 |
| **Wiki / 위키** | 📓 | ドキュメント、コレクション、検索 / 문서, 컬렉션, 검색 |
| **設定 / 설정** | ⚙️ | 言語、テーマ、アカウント情報 / 언어, 테마, 계정 정보 |

各タブは独立して状態を保持します。タブを切り替えても、前の画面の位置やスクロール状態は保持されます。

각 탭은 독립적으로 상태를 유지합니다. 탭을 전환해도 이전 화면의 위치와 스크롤 상태가 유지됩니다.

## 4. 在庫タブ（InvenTree） / 재고 탭 (InvenTree)

### 概要 / 개요

在庫タブはInvenTreeシステムと連携し、部品・在庫・注文を管理します。

재고 탭은 InvenTree 시스템과 연동하여 부품·재고·주문을 관리합니다.

### ホーム画面 / 홈 화면

在庫タブを開くと、以下のメニューカードが表示されます:

재고 탭을 열면 다음 메뉴 카드가 표시됩니다:

| カード / 카드 | 説明 / 설명 |
|---|---|
| **部品 / 부품** | 部品一覧の閲覧、カテゴリ別ブラウズ / 부품 목록 조회, 카테고리별 탐색 |
| **在庫 / 재고** | 在庫アイテムの確認、保管場所別表示 / 재고 아이템 확인, 보관 장소별 표시 |
| **発注 / 발주** | 仕入先への発注書（Purchase Orders）/ 공급업체 발주서 |
| **受注 / 수주** | 顧客からの受注（Sales Orders）/ 고객 수주 |
| **送り状 / 운송장** | 送り状の生成と印刷 / 운송장 생성 및 인쇄 |

画面下部には「未処理注文サマリー」が表示され、発注・受注の件数を確認できます。

화면 하단에는 "미처리 주문 요약"이 표시되며, 발주·수주 건수를 확인할 수 있습니다.

### 4.1 部品の閲覧 / 부품 조회

1. **「部品」** カードをタップ / **"부품"** 카드 클릭
2. 部品一覧が表示される / 부품 목록이 표시됨
3. 上部の検索バーで部品名を検索可能 / 상단 검색창에서 부품명 검색 가능
4. 部品をタップすると詳細画面へ / 부품을 클릭하면 상세 화면으로 이동

#### 部品詳細画面 / 부품 상세 화면

- **基本情報**: 部品名、説明、カテゴリ、在庫数
- **기본 정보**: 부품명, 설명, 카테고리, 재고수
- サムネイル画像がある場合は表示されます / 썸네일 이미지가 있으면 표시됩니다

### 4.2 在庫の確認 / 재고 확인

1. **「在庫」** カードをタップ / **"재고"** 카드 클릭
2. 在庫アイテム一覧が表示される / 재고 아이템 목록이 표시됨
3. 各アイテムには数量と保管場所が表示される / 각 아이템에는 수량과 보관 장소가 표시됨
4. アイテムをタップすると詳細画面へ / 아이템을 클릭하면 상세 화면으로 이동

#### 在庫詳細画面 / 재고 상세 화면

- **数量**: 現在の在庫数 / 현재 재고수
- **保管場所**: 倉庫・棚の場所 / 창고·선반 위치
- **部品情報**: 該当部品へのリンク / 해당 부품으로 링크
- **シリアル番号**: 設定されている場合 / 설정되어 있는 경우
- **ステータス**: 在庫の状態（正常、返品中等）/ 재고 상태 (정상, 반품 중 등)

### 4.3 発注管理 / 발주 관리

1. **「発注」** カードをタップ / **"발주"** 카드 클릭
2. 発注書一覧が表示される（未処理のものが優先表示）/ 발주서 목록 표시 (미처리 우선)
3. 各発注書には参照番号、仕入先名、ステータスが表示される / 각 발주서에는 참조번호, 공급업체명, 상태가 표시됨

#### ステータス / 상태

| ステータス / 상태 | 意味 / 의미 |
|---|---|
| **Pending** | 保留中 / 보류 중 |
| **Placed** | 発注済み / 발주 완료 |
| **Complete** | 入荷完了 / 입고 완료 |
| **Cancelled** | キャンセル / 취소 |

### 4.4 受注管理 / 수주 관리

1. **「受注」** カードをタップ / **"수주"** 카드 클릭
2. 受注一覧が表示される / 수주 목록이 표시됨
3. 各受注には参照番号、顧客名、ステータスが表示される / 각 수주에는 참조번호, 고객명, 상태가 표시됨

### 4.5 送り状（Waybill） / 운송장 (Waybill)

送り状の生成はInvenTree Invoice Printプラグインと連携します。

운송장 생성은 InvenTree Invoice Print 플러그인과 연동됩니다.

1. **「送り状」** カードをタップ / **"운송장"** 카드 클릭
2. 送り状生成画面が表示される / 운송장 생성 화면이 표시됨
3. **「送り状を生成」** ボタンをクリック / **"운송장 생성"** 버튼 클릭
4. ジョブIDが表示されたら生成成功 / 작업 ID가 표시되면 생성 성공
5. プリンターから自動で出力される / 프린터에서 자동으로 출력됨

**対応運送会社 / 지원 운송회사**: ヤマト運輸、佐川急便 / 야마토운수, 사가와큐빈

## 5. タスクタブ（Vikunja） / 작업 탭 (Vikunja)

### 概要 / 개요

タスクタブはVikunjaと連携し、プロジェクト・タスクを管理します。

작업 탭은 Vikunja와 연동하여 프로젝트·업무를 관리합니다.

### 5.1 プロジェクト一覧 / 프로젝트 목록

タスクタブを開くと、プロジェクト一覧が表示されます。

작업 탭을 열면 프로젝트 목록이 표시됩니다.

- 各プロジェクトにはカラーアイコン、名前、説明が表示される / 각 프로젝트에는 컬러 아이콘, 이름, 설명이 표시됨
- アーカイブされたプロジェクトにはアーカイブアイコンが付く / 아카이브된 프로젝트에는 아카이브 아이콘이 표시됨
- プロジェクトをタップするとタスク一覧へ / 프로젝트를 클릭하면 업무 목록으로 이동

### 5.2 プロジェクト詳細（タスク一覧） / 프로젝트 상세 (업무 목록)

プロジェクトを選択すると、そのプロジェクトのタスク一覧が表示されます。

프로젝트를 선택하면 해당 프로젝트의 업무 목록이 표시됩니다.

#### 表示モード / 표시 모드

画面上部のアイコンで表示モードを切り替えられます:

화면 상단 아이콘으로 표시 모드를 전환할 수 있습니다:

| モード / 모드 | 説明 / 설명 |
|---|---|
| **リスト / 리스트** | タスク一覧（デフォルト）/ 업무 목록 (기본값) |
| **カンバン / 칸반** | ドラッグ＆ドロップでステータス変更可能 / 드래그앤드롭으로 상태 변경 가능 |
| **カレンダー / 캘린더** | 期限日をカレンダーに表示 / 기한일을 캘린더에 표시 |

### 5.3 カンバンボード / 칸반 보드

カンバンボードでは、タスクをバケット（列）間でドラッグして状態を変更できます。

칸반 보드에서는 업무를 버킷(열) 사이에서 드래그하여 상태를 변경할 수 있습니다.

- 各カードにはタスクタイトル、優先度マーク、期限が表示される / 각 카드에는 업무 제목, 우선도 표시, 기한이 표시됨
- カードをタップするとタスク詳細画面へ / 카드를 클릭하면 업무 상세 화면으로 이동

### 5.4 カレンダー表示 / 캘린더 표시

カレンダー表示では、期限付きタスクが日付上に表示されます。

캘린더 표시에서는 기한이 있는 업무가 날짜 위에 표시됩니다.

- 月表示 / 주표示 / 2週間表示を切り替え可能 / 월/주/2주 표시 전환 가능
- 日付をタップするとその日のタスク一覧が下部に表示される / 날짜를 클릭하면 해당 날짜의 업무 목록이 하단에 표시됨

### 5.5 タスク詳細 / 업무 상세

タスクをタップすると詳細画面が表示されます:

업무를 클릭하면 상세 화면이 표시됩니다:

- **タイトル / 제목**: タスク名 / 업무명
- **説明 / 설명**: タスクの詳細説明 / 업무 상세 설명
- **優先度 / 우선도**: 緊急、高、中、低、なし / 긴급, 높음, 중간, 낮음, 없음
- **期限 / 기한**: 期限日時 / 기한 일시
- **担当者 / 담당자**: 割り当てられたユーザー / 배정된 사용자
- **ラベル / 라벨**: カテゴリ別の色付きラベル / 카테고리별 컬러 라벨
- **完了 / 완료**: チェックマークで完了/未完了を切り替え / 체크 표시로 완료/미완료 전환

### 5.6 タスクの作成 / 업무 생성

プロジェクト詳細画面の **「+」** ボタンをタップしてタスクを作成できます。

프로젝트 상세 화면의 **"+"** 버튼을 클릭하여 업무를 생성할 수 있습니다.

入力項目:
- タイトル（必須）/ 제목 (필수)
- 説明 / 설명
- 優先度 / 우선도
- 期限日 / 기한일
- ラベル / 라벨

## 6. Wikiタブ（Outline） / 위키 탭 (Outline)

### 概要 / 개요

WikiタブはOutlineと連携し、社内ドキュメントを管理します。

위키 탭은 Outline과 연동하여 사내 문서를 관리합니다.

### 6.1 ホーム画面 / 홈 화면

Wikiタブを開くと、以下が表示されます:

위키 탭을 열면 다음이 표시됩니다:

- **コレクション / 컬렉션**: フォルダのようにドキュメントを整理 / 폴더처럼 문서를 정리
- **最近のドキュメント / 최근 문서**: 最近更新されたドキュメント / 최근 수정된 문서

上部の検索アイコンをタップすると全文検索が可能です。

상단의 검색 아이콘을 클릭하면 전문 검색이 가능합니다.

### 6.2 コレクション / 컬렉션

コレクションはフォルダのような役割です。タップするとそのコレクションに属するドキュメント一覧が表示されます。

컬렉션은 폴더 같은 역할입니다. 클릭하면 해당 컬렉션에 속하는 문서 목록이 표시됩니다.

### 6.3 ドキュメントの閲覧 / 문서 조회

ドキュメントをタップすると、Markdown形式で表示されます。

문서를 클릭하면 Markdown 형식으로 표시됩니다.

- 見出し、箇条書き、テーブル、リンクなどが正しくレンダリングされます
- 제목, 목록, 테이블, 링크 등이 올바르게 렌더링됩니다

### 6.4 ドキュメントの編集 / 문서 편집

ドキュメント詳細画面の編集ボタンをタップすると、エディター画面が開きます。

문서 상세 화면의 편집 버튼을 클릭하면 에디터 화면이 열립니다.

- Markdown形式で編集可能 / Markdown 형식으로 편집 가능
- 保存ボタンでOutlineに直接保存される / 저장 버튼으로 Outline에 직접 저장됨

### 6.5 検索 / 검색

1. Wiki画面右上の検索アイコンをタップ / 위키 화면 우측 상단 검색 아이콘 클릭
2. キーワードを入力 / 키워드 입력
3. 検索結果にはドキュメントタイトルとマッチしたテキストが表示される / 검색 결과에 문서 제목과 매치된 텍스트가 표시됨
4. 結果をタップすると該当ドキュメントに遷移 / 결과를 클릭하면 해당 문서로 이동

## 7. 設定タブ / 설정 탭

### 7.1 アカウント情報 / 계정 정보

画面上部に現在ログインしているユーザーの以下の情報が表示されます:

화면 상단에 현재 로그인한 사용자의 다음 정보가 표시됩니다:

- プロフィール写真（Googleアカウントのもの）/ 프로필 사진 (Google 계정의 것)
- 表示名 / 표시 이름
- メールアドレス / 이메일 주소

### 7.2 言語設定 / 언어 설정

3つの言語から選択できます:

3개 언어에서 선택할 수 있습니다:

## 8. 日常業務ワークフロー / 일상 업무 워크플로우

### 8.1 朝の業務開始 / 아침 업무 시작

1. `app.your-domain.com` にアクセスしてログイン / 접속 후 로그인
2. **在庫タブ** → 未処理注文サマリーを確認 / **재고 탭** → 미처리 주문 요약 확인
3. **タスクタブ** → 今日のタスクをカレンダーで確認 / **작업 탭** → 오늘의 업무를 캘린더에서 확인
4. 優先度の高いタスクから着手 / 우선도가 높은 업무부터 착수

### 8.2 注文処理フロー / 주문 처리 플로우

1. **在庫タブ** → **受注** を開く / **재고 탭** → **수주** 열기
2. 新しい受注を確認 / 새 수주 확인
3. 部品の在庫を確認（在庫タブ → 在庫）/ 부품 재고 확인 (재고 탭 → 재고)
4. **送り状** を開いて送り状を生成 / **운송장** 열어서 운송장 생성
5. プリンターで送り状を出力 / 프린터에서 운송장 출력
6. 出荷後、在庫は自動更新 / 출하 후 재고 자동 업데이트

### 8.3 タスク管理フロー / 업무 관리 플로우

1. **タスクタブ** → プロジェクトを選択 / **작업 탭** → 프로젝트 선택
2. 新しいタスクを **「+」** ボタンで追加 / 새 업무를 **"+"** 버튼으로 추가
3. 優先度と期限を設定 / 우선도와 기한 설정
4. **カンバン** で進捗を管理（ドラッグ＆ドロップ）/ **칸반**에서 진행 상황 관리 (드래그앤드롭)
5. 完了したらチェックマークをタップ / 완료하면 체크 표시 클릭

### 8.4 マニュアル参照 / 매뉴얼 참조

1. **Wikiタブ** → コレクションから該当カテゴリを選択 / **위키 탭** → 컬렉션에서 해당 카테고리 선택
2. または検索アイコンでキーワード検索 / 또는 검색 아이콘으로 키워드 검색
3. 手順書を確認しながら業務を進める / 절차서를 확인하면서 업무 진행

### 8.5 発注処理フロー / 발주 처리 플로우

1. **在庫タブ** → **部品** で在庫が少ない部品を確認 / **재고 탭** → **부품**에서 재고가 적은 부품 확인
2. **発注** を開いて発注書を確認 / **발주** 열어서 발주서 확인
3. 仕入先に発注（InvenTreeのWebインターフェースで実施）/ 공급업체에 발주 (InvenTree 웹 인터페이스에서 실시)
4. 入荷時にステータスを更新 / 입고 시 상태 업데이트

## 9. トラブルシューティング / 문제 해결

### ログインできない場合 / 로그인이 안 될 때

| 症状 / 증상 | 対処 / 해결 |
|---|---|
| Googleアカウント選択後にエラー | `@your-domain.com` のアカウントを使用しているか確認 / `@your-domain.com` 계정 사용 확인 |
| 画面が真っ白 | ブラウザのキャッシュをクリアして再読み込み / 브라우저 캐시 삭제 후 새로고침 |
| 「ログインエラー」表示 | ネットワーク接続を確認。VPN接続が必要な場合あり / 네트워크 연결 확인. VPN 접속이 필요할 수 있음 |

### データが表示されない場合 / 데이터가 표시되지 않을 때

| 症状 / 증상 | 対処 / 해결 |
|---|---|
| 在庫一覧が空 | InvenTreeサーバーの稼働状態を確認 / InvenTree 서버 가동 상태 확인 |
| タスクが表示されない | Vikunjaサーバーの稼働状態を確認 / Vikunja 서버 가동 상태 확인 |
| Wikiが空 | Outlineサーバーの稼働状態を確認 / Outline 서버 가동 상태 확인 |
| 「Error」表示 | 画面を下に引いて更新（Pull-to-refresh）/ 화면을 아래로 당겨서 갱신 |

### その他 / 기타

| 症状 / 증상 | 対処 / 해결 |
|---|---|
| 言語が変わらない | 設定タブで言語を選択し直す / 설정 탭에서 언어를 다시 선택 |
| 画面が崩れる | ブラウザを最新版に更新 / 브라우저를 최신 버전으로 업데이트 |
| 動作が遅い | ブラウザのタブを減らす / 브라우저 탭 수를 줄이기 |
