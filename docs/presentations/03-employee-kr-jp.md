# 주문 처리 / 注文処理

## 이전 / Before

1. 메일에서 주문 확인 → メールで注文確認
2. 엑셀에서 재고 확인 → Excelで在庫確認
3. 운송회사 사이트에서 운송장 작성 → 配送サイトで送り状作成
4. 엑셀에서 재고 수정 → Excelで在庫修正

## 이후 / After

1. 대시보드에서 주문 확인 → ダッシュボードで注文確認
2. **확인 버튼 클릭** → **確認ボタンをクリック**
3. 운송장 자동 생성 → 送り状が自動作成
4. 재고 자동 업데이트 → 在庫が自動更新

> **클릭 몇 번이면 끝! / 数クリックで完了！**

# 운송장 출력 / 送り状印刷

## 새로운 방법 / 新しい方法

1. 대시보드 "재고" 탭 열기
   ダッシュボードの「在庫」タブを開く

2. 주문 선택
   注文を選択

3. "운송장 출력" 버튼 클릭
   「送り状印刷」ボタンをクリック

4. 프린터에서 자동으로 출력됨
   プリンターから自動で印刷される

- 야마토, 사가와 모두 지원
  ヤマト、佐川どちらも対応

- 주소 직접 입력 불필요
  住所の手入力不要
