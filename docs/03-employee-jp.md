# 毎日の仕事がどう変わりますか？

## 一言で: もっと簡単になります

- 複数のサイトに別々にログインする必要がなくなります
- Googleアカウント一つで全てアクセス
- ダッシュボード一つで全ての業務を確認
- 分からないことはAIが教えてくれます

# 注文処理

## Before

1. メールで注文確認
2. Excelで在庫確認
3. 配送サイトで送り状作成
4. Excelで在庫修正

## After

1. ダッシュボードで注文確認
2. **確認ボタンをクリック**
3. 送り状が自動作成
4. 在庫が自動更新

> **数クリックで完了！**

---

# 送り状印刷

## 新しい方法

1. ダッシュボードの「在庫」タブを開く
2. 注文を選択
3. 「送り状印刷」ボタンをクリック
4. プリンターから自動で印刷される

- ヤマト、佐川どちらも対応
- 住所の手入力不要

# タスク管理

## やるべきことを忘れない

### Vikunjaタスク管理

- **カンバンボード**: カードをドラッグして状態変更
- **カレンダー**: 期限がカレンダーに表示
- **担当者割り当て**: 誰の仕事か明確に

# 始めましょう

## まずやること

1. **Googleアカウント確認** — 会社メールでログインします
2. **ダッシュボードにアクセス** — `app.your-domain.com`
3. **「Googleでログイン」ボタンをクリック**
4. **言語設定** — 設定タブで韓国語/日本語/英語を選択

> 難しければAIアシスタントに聞いてください！
