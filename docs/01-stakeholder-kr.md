# 1. 현재의 과제

## 업무상의 문제점

| 과제 | 영향 |
|------|------|
| Excel로 재고 관리 → 입력 실수, 중복 | 출하 지연, 재고 불일치 |
| 각 운송회사 사이트에 개별 로그인 | 시간 낭비, 비밀번호 관리 번거로움 |
| 사내 지식이 개인에 의존 | 퇴직 시 업무 지식 상실 |
| 업무 관리가 구두 기반 | 누락, 중복 작업 |
| 마켓플레이스 주문 수동 처리 | 처리 지연, 오류 |

# 3. 업무 플로우 변화

## 이전

1. 주문 수신 → 메일 확인 → 수동으로 재고 확인 → Excel에 기입
2. 운송장 작성 → 야마토 사이트 로그인 → 수동 입력 → 인쇄
3. 재고 업데이트 → Excel 수동 수정 → 공유 드라이브에 업로드
4. 정보 공유 → 구두, 메모, LINE

## 이후

1. 주문 수신 → **자동 가져오기** → 대시보드에 표시 → 원클릭 승인
2. 운송장 작성 → InvenTree **플러그인이 자동 생성** → 인쇄 버튼만
3. 재고 업데이트 → **실시간 자동 업데이트** → 모든 직원이 같은 데이터 확인
4. 정보 공유 → **Wiki + 업무 관리** → 검색 가능, 이력 보존

# 5. AI 업무 지원

## AI 어시스턴트의 역할

**조작 가이드** — 화면을 보면서 단계별로 안내
- "이 주문은 어떻게 처리하면 되나요?"
- "운송장 인쇄 방법을 알려주세요"

**업무 관리** — 자연어로 업무 조작
- "다음 주 야마토 집하를 업무에 추가해줘"
- "사가와 미완료 업무를 보여줘"

**회의 기록** — 음성에서 자동으로 업무 추출
- 회의 녹음 → 텍스트 변환 → 업무 후보 추출 → 승인 후 Vikunja에 반영

**개인정보 보호**: 모든 데이터는 AI에 보내기 전에 마스킹 처리됩니다
